# Pro Stock Market Analysis System 📈

A professional-grade web application for real-time stock market analysis, built with **Streamlit**, **yFinance**, and **Plotly**.

## 🚀 Key Features
*   **Real-time Data Fetching**: Seamless integration with Yahoo Finance API.
*   **Advanced Visualizations**:
    *   Interactive **Candlestick** and **Line** charts.
    *   Integrated **Volume** analysis.
*   **Technical Indicators**: On-the-fly calculation of Moving Averages (SMA 20, SMA 50).
*   **Performance Metrics**: Real-time display of current price, period changes, and high/low values.
*   **Company Insights**: Comprehensive company profiles, including sector, industry, and market cap.
*   **Optimized Performance**: Data caching to reduce API calls and improve loading speed.

## 🛠️ Tech Stack
*   **Frontend**: Streamlit
*   **Data Source**: yFinance (Yahoo Finance API)
*   **Data Processing**: Pandas
*   **Visualization**: Plotly (Interactive Charts)

## 📦 Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone <your-repo-url>
   cd stock_analysis_project
   ```

2. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the application**:
   ```bash
   streamlit run app.py
   ```
