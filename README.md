# Stock Market Analysis System 📈
This is a simple web application built with **Streamlit** and **Python** to analyze stock market trends using **Yahoo Finance API**.

## Features
* Search for any stock symbol (e.g., AAPL, TSLA).
* View interactive price charts.
* Display historical data tables.

## Tech Stack
* Python
* Streamlit
* yFinance API
* Plotly for Visualization
